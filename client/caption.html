AttnGAN

Instructions:
  • Select model
  • Pick example caption
  • Hit generate
    • You can make changes to the example caption (for example, change color, beak length etc). The images are automatically generated by an AI algorithm called "AttentionGAN". The generation is probabilistic i.e., if you hit generate multiple times for the same caption, the generated images will be different (but will reflect the information provided by your caption)

    AttentionGAN

    This demo is based on the Pytorch implementation of AttnGAN model described in Fine-Grained Text to Image Generation with Attentional Generative Adversarial Networks by Tao Xu, Pengchuan Zhang, Qiuyuan Huang, Han Zhang, Zhe Gan, Xiaolei.
